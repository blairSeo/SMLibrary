## Installation

SMLibrary is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'SMLibrary'
```
And then run:

```ruby
pod install
```

And installing the cocoapod into your project import SMLibrary with

```swift
import SMLibrary 
```
